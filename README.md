## Introduction to Parsing

This repository is a public educational resource for learning the basics of parsing. It is designed as a set of small, focused projects that demonstrate how parsers are built and tested in practice.

Each project isolates a key concept so you can understand parsing step by step.

---

## What this repo contains

Each folder focuses on a specific part of parsing:

- **Project 1 - basic_parsing** – Simple parsing concepts and string processing  
- **Project 2 - tokenization** – Breaking input into tokens  
- **Project 3 - grammar_parsing** – Using grammar rules to structure input  
- **Project 4 - expression_parsing** – Parsing arithmetic-style expressions  
- **Project 5 - project_examples** – Small applied parsing examples  

---

## Purpose

Parsing is a core concept in programming languages and compilers, but it is often introduced in a complex way.

This repository is meant to make it easier by:
- Breaking parsing into small steps  
- Showing working implementations  
- Giving simple, testable examples
